About the Project
This 3D render shows a minimalistic clean workspace. I created this render as part of a module in the third semester. I had a lot of freedom with this project because the task was just, to create a room that shows a workspace. I used blender for this project and modeld all the object by myself. The textures I took from polyhaven.com. I learnd a lot about modelling and composing the scene in this project.
